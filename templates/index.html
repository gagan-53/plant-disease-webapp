    Analyzing leaf…

    1. Preprocess

    Image resized to 128×128, converted to RGB, and normalized to [0, 1] — the same pipeline used during training.

    2. CNN inference

    3 stacked convolution + pooling blocks (32 → 64 → 128 filters), flattened, then a 128-unit dense layer with 0.5 dropout before softmax.

    3. Diagnose

    Softmax probabilities are returned across all classes. Top prediction surfaces plant species, disease, and a recommended action.

    Model architecture

    Input(128, 128, 3)
  └─ Conv2D(32, 3×3, ReLU)  → MaxPool(2×2)
  └─ Conv2D(64, 3×3, ReLU)  → MaxPool(2×2)
  └─ Conv2D(128, 3×3, ReLU) → MaxPool(2×2)
  └─ Flatten
  └─ Dense(128, ReLU) → Dropout(0.5)
  └─ Dense(N, Softmax)
